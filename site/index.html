acet
Same diamond · Different face

A vector-based resume engine. Maintain one source of truth, define targeting vectors for each role type, and generate pixel-perfect PDFs that show the right facet of your experience — every time.

The idea
01

One diamond

All your experience, skills, and accomplishments live in a single structured data source. No copy-pasting between files.

02

Many facets

Define vectors — targeting profiles that control which bullets surface, what order they appear, and how skills are weighted.

03

Cut to fit

Generate a tailored resume for each opportunity. Same truth, different face. Every version is consistent, current, and precise.

Features
Core

Vector Targeting

Per-vector bullet ordering, priority overrides, and skill group routing. Each vector is a lens that reveals the most relevant slice of your experience.

Render

Typst PDF Engine

WASM-powered Typst rendering produces pixel-perfect PDFs with bundled fonts. True WYSIWYG preview — what you see is what gets downloaded.

Design

Theme System

8 built-in presets with per-token overrides for fonts, spacing, margins, colors, and layout. Density controls for one-step global tightening or loosening.

AI

JD Analysis

Paste a job description, get targeted suggestions for vector tuning, bullet reordering, and priority shifts. Apply selectively and save as a new variant.

Data

Import / Export

Round-trippable JSON with replace and merge modes. Schema validation with actionable error reporting. Your data is always portable.

State

Saved Variants

Named presets with dirty-state tracking. Save your carefully tuned configurations and switch between them instantly.

Workflow
01

Load your data

02

Define vectors

Create targeting profiles — platform eng, backend, security, DevOps. Each one surfaces different content.

03

Reorder bullets, adjust priorities, pick a theme. WYSIWYG preview shows the real PDF in real time.

04

Export

Download pixel-perfect PDF, copy plain text for forms, or export Markdown. Save the config as a preset.
